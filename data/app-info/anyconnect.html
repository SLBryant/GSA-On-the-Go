The Cisco AnyConnect app allows you to connect your mobile device to the GSA network to securely access GSA resources including InSite, Chatter, and Salesforce.

Configuring the App

  1. Open the AnyConnect app and select OK in the dialog box that asks if you would like to enable the software
  2. Tap Add Connection
  3. Enter a Description e.g. GSA VPN
  4. Enter the Server Address: vpn.gsa.gov/mobile and Save
