SecureAuth OTP generates a one-time password which authenticates you when you use mobile device VPN.

Configuring the App

  1. Open the SecureAuth OTP application and enter secureauth.gsa.gov for the Server URL
  2. Enter your ENT username and select a delivery method for a one-time registration code
  3. Enter the 4 digit registration code and hit Submit
  4. Enter your ENT password and hit Submit
  5. Set a 4 digit PIN of your choice to use whenever you access the app
